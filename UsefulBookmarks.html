Useful Bookmarks

Site Description
W3 Schools This is the website we used to learn HTML and CSS. It's an amazing resource and has documentation for basically everything.
Replit Replit is the site we used before the school board decided we should switch to Github. We did the first two units of Python on this site, and it was great because it was organized and easy to read.
Firewall Replit This is a firewalled version of Replit, meaning we could access it offline. Initially, we had really bad Wi-fi in class, so we used this for about a week.
CS Circles We used CS Circles to learn Python. It has exercises that are helpful to do.
CMU CMU is a website that taught us Python to make graphics. We had multiple holiday challenges and we got to the end of Unit 2.
Github Github is the code editor we used for our portfolios and our website project. It's similar to Replit.
CodeHS This is the site we used for exercises and assignments after we were kicked out of Replit. It's very well orgaized and easy to use. The code checker is very specific, which can be frustrating at times.
